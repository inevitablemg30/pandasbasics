## Sales Data Analysis using Pandas
This project analyzes a small sales dataset containing information about ten salespersons. The dataset includes details such as region, product sold, units sold, and total sales amount for the month of January. The goal is to perform basic exploratory data analysis using Python and Pandas.

## Dataset Description
The CSV file contains the following columns:
1. Person_ID: Unique ID for each salesperson
2. Name: Name of the salesperson
3. Region: Sales region (North, South, East, West)
4. Product: Product sold
5. Units_Sold: Number of units sold
6. Sales_Amount: Total revenue generated
7. Month: Month of sale

## Features of the Project
Load the dataset using Pandas
View initial rows and dataset structure
Calculate total sales
Group data by region and product
Identify top performing salesperson
Perform summary statistics such as mean, max, and sum

## How to Run
1. Install Python
2. Install Pandas using
3. pip install pandas
4. Place your CSV file in the project directory
5. Run the analysis script using
6. python analyze.py
